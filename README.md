### Methods

#### Vehicle Id and number of n clusters are taken as parameters on the clustering method. Vehicle 1 has 9 containers. The method is called via Swagger so that n=4 and VehicleId = 1. The vehicle first filled and unloaded 4 times starting from the closest containers. Since the clustering is done according to the distance, it saves time.
